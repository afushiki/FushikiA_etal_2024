# A study on Parkinson's disease model mice
Github repo for the paper "A Vulnerable Subtype of Dopaminergic Neurons Drives Early Motor Deficits in Parkinson’s Disease". A draft of the paper is available as a preprint on bioRxiv, which can be accessed [here](https://www.biorxiv.org/content/10.1101/2024.12.20.629776v1).

## **Contents**
### Data
- Single-cell RNA-sequencing data is deposited in the Gene Expression Omnibus website ([GSE285508](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE285508)). 

### Scripts
- [Sequencing Analysis](./Scripts_for_sequencing_analysis)
